# Parko [RU]
Сервис для учёта автопарка предприятий (SaaS) с поддержкой множества компаний.
Backend на Django REST Framework обеспечивает управление машинами, расходом топлива, ТО, страховками и отчётами, с ролевой моделью доступа (админ, диспетчер, механик, бухгалтер, гость).
В дальнейшем планируется React‑frontend и AI‑ассистент для генерации отчётов и работы с данными в формате диалога.
