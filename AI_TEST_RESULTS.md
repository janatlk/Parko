## ⚠️ ПРОБЛЕМНЫЕ МЕСТА И РЕШЕНИЯ

### Проблема 1: Пустая таблица топлива (image.png)
**Запрос:** "Сколько топлива было потрачено в этом месяце?"  
**Проблема:** ⚠️ Таблица пустая (колонки "-", "-", "**13**")  
**Ожидание:** Полная таблица с данными

**Диагноз:**
- AI получил правильные данные (13 литров)
- Но таблица сформирована неправильно - колонки пустые
- Проблема в промпте: AI не получает правильную структуру для таблицы

**Причина:** В `collect_company_context()` данные собираются агрегированно, а не построчно. AI пытается создать таблицу но нет детальных данных.

**Решение:** Улучшить промпт или передавать детальные данные в контекст.

---

---

### Проблема 4: Сырой JSON в ответе (image copy 5.png)
**Запрос:** "Какие страховки скоро истекают?"  
**Проблема:** ⚠️ Показан сырой JSON вместо таблицы:  
```json
{ "type": "table", "headers": [...], "rows": [] }
```

**Диагноз:**
- AI сгенерировал JSON для таблицы
- Но таблица пустая (rows: [])
- MarkdownRenderer НЕ отрендерил JSON в таблицу
- Проблема: JSON показан как текст, не как таблица

**КРИТИЧЕСКАЯ ПРОБЛЕМА:** MarkdownRenderer не парсит JSON таблицы когда они пустые или в начале ответа!

**Решение:** 
1. Проверить MarkdownRenderer - должен рендерить JSON таблицы
2. AI должен не показывать JSON если таблица пустая
3. Добавить проверку в парсер

```markdown

```

Но иногда AI возвращает JSON без markdown блока.

**Что проверить:**
1. Функция `parseActionsFromContent` - должен извлекать JSON таблицы

3. Добавить обработку JSON таблиц вне code blocks
